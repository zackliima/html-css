Código-Fonte / Pré-Formtação

O comando document.getElementById('teste') é escrito em linguagem JavaScript.
Assim, a tag <code> deixa representado ao navegador que tudo que estiver dentro dessa tag, deverá ficar em formado de código.
Observe o seguinte exemplo: 

        
            

                num = int(input(Digite um Número:))
                if num % 2 == 0;
                    print(f'O Número{num} é PAR')
                else:
                    print(f'O Número{num} é IMPAR')
                print('Fim do Programa')

Viu? Repare que a tag <pre> mantém a formatação e identação original, da exata maneira que ela foi escrita. Enquanto a <code> deixa tudo em formado de código no site.
