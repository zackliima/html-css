Principais Formtações

Negrito / Destaque

Nesta frase, temos um termo em negrito usando a tag <b> (não Semântico).

Nesta frase, temos um termo em negrito usando a tag <strong> (semântico).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (não semântico).

Nesta frase, temos um termo em itálico usando a tag <em> (semântico).

Texto Marcado

Também podemos criar um texto marcado usando a tag <mark> (semântico).

Mas também podemos marcar um texto em outro parágrafo (<p>), utilizando o <style> no header a marcando a tag <mark> como parâmetro.

Texto Grande e Pequeno

Estamos utilizando um texto grande utilizado com a tag <big> (não semântico).
E um texto pequeno utilizado com a tag <small> (semântico).

Acredita-se que não existe a necessidade da tag <big> nos dias atuais, devido as formas/maneiras que o CSS já proporcionar para isso. No entanto, a tag <small> ainda pode ser utilizada devido ao fato de que em contratos e outros demais textos formais, ainda possui textos importantes que ficam pequenos mas que ainda tem significado!

Texto Deletado

Podemos marcar um texto excluído que ainda pode ser lido, mas não considerado para o conteúdo apresentado.

Texto Inserido

Também tem a possibilidade de inserir um texto no conteúdo com a tag <ins>, indicando que ele foi inserido depois (semântico)

Existe também o sublinhado com a tag <u> (não semântico)

Texto Sobrescrito e Subscrito

Existe formas de inserir coisas do tipo 219x3 (expressão matemática), utilizado com a tag <sup>.
E inserir também a fórmula da água por exemplo: H2O, utilizado com a tag <sub>.
Tag <sub> = Usado para subscrever algo no texto.
Tag <sup> = Usado para sobrescrever algo no texto.
